# How to Contribute to This Repository

## Step 1: Clone the Repository (First Time Only)

```bash
git clone git@github.com:UEM-Kolkata-2023-27/ContextBasedRAG.git
cd ContextBasedRAG

Iniatialize yourself using you maail and name ->
git config user.name "Your Full Name"
git config user.email "your-email@example.com"  (Email linked to GitHub)



## Step 2: Create Your Own Branch

git checkout main
git pull origin main
git checkout -b your-name-feature


git checkout your-branch
git merge main

git checkout your-branch
git rebase main



## Step 3: Make Changes and Commit
git add .
git commit -m "Describe your changes"


## Step 4: Push Your Branch
git push origin your-name-feature

##Step 5: Create a Pull Request (PR)
Go to the repository on GitHub
Click Pull Requests
Click New Pull Request
Base branch: main
Compare branch: your-name-feature
Click Create Pull Request

## Step 6: Wait for Approval and Merge
The project owner will review your PR
After approval, the PR will be merged into main

## Step 7: Update Your Branch If main Changes
git checkout main
git pull origin main
git checkout your-name-feature
git merge main


## Rules   IMPORTANT
Do NOT push to main
Always work in your own branch
All changes must go through Pull Requests
Do NOT run (git init)


❌ Common Mistakes (DO NOT DO THESE)
❌ Running git init
❌ Pushing directly to main
❌ Working without creating a branch
❌ Using random branch names like test, new, final
❌ Force pushing
❌ Committing without setting name and email
```
